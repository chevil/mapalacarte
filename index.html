Help

This is a tool to create your own map, in the traditional sense of the term, but not with tiles, mas bien with layers rendered in WebGL using the code of the MapLibre GL JS project.

You have 3 basic themes here, provided by Map Libre GL, and when you load the page, one theme is already selected for you. You can change the theme you want to work with by clicking the radio theme box.

In the selection menu below, you have the list of layers of the theme you chose, it is different for each theme provided. When loading a theme, all the layers are hidden by default.

When you select a layer, a window opens with all the CSS-like parameters of that layer that you can set for this layer. You can set the values as you like and click on 'Set' and your layer should appear with the values that you have set.

If you set wrong parameters, do not worry as your values will not be considered and you should try again.

If you don't see your layer, remember that some layers are only visible at certain zoom values, so try to zoom in and out to make them appear.

Common layers you should set to see your map are things like : water, park, label_city, boundary_2 (frontiers), ...

Don't get intimated with parameters starting with 'interpolate, ...', it usually means the value is changing according to the zoom, but you can enter a simple value as well.

Once all the layers you wish are configured, you can save your map, and you will be pointed to a simple version of your map to start your project with.
